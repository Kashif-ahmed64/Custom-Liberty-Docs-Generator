# Liberty Pass Generator

Reusable web tool for generating printable lanyard passes for **Liberty Daharki Power Ltd.** events.

## Official Branding

Matches the official letterhead format:

- **Circular maroon logo** with Urdu calligraphy
- **LIBERTY** <span style="color:#cc0000">**DAHARKI**</span> **POWER LTD.** — split black/red typography
- **Blue facility lines** — plant name and location
- **Date format** — `(dt: 17.06.2026)`
- **White background** for print-friendly output
- **Card size:** 75mm × 100mm (fits OLD ID lamination pouches)

## Features

- All fields live-editable (company name, facility lines, role tag, name, date)
- **+ Company Name** block — official split-color heading
- **+ Text / Logo / Tag Box** — add custom fields with formatting
- 6 theme templates (Liberty Official default)
- Print & PNG download — white card, official colors preserved

## Default Template

| Zone | Content |
|------|---------|
| Header | Circular logo, LIBERTY DAHARKI POWER LTD., plant & location (blue) |
| Body | INVIGILATOR / Trade Apprentice Test + Name |
| Footer | (dt: 17.06.2026) |
